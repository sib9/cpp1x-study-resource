###cpp11-study-resource###

期盼已久的C++新标准(C++0x,C++11)于2011年8月份获得一致通过。时隔近两年，主流的C++编译器(GCC,VS2012,Clang)已经支持了很多C++的新特性。[GCC 4.8.1](http://gcc.gnu.org/gcc-4.8/cxx0x_status.html)已实现C++11标准的所有主要语言特性。

####目的####
1. 搜集C++11资源，供大家参考、学习;
2. 学习并使用C++11的新特性来写C++程序;

####组成####
1. 资料(来源于文档、博客、论坛等);
2. 原理分析(知其然，还要知其所以然);
3. 对应特性的测试代码;
